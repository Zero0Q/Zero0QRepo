Zero0Q Kodi Repository

Welcome to the Zero0Q Kodi Repository. This repository contains the GTV skin and related addons for a Google TV-like experience in Kodi.

Installation Methods

Method 1: Quick Install

Click the button below to install the repository in Kodi:

Install Repository

Method 2: Manual Installation

  1. Download repository.zero0qrepo-0.0.4.zip
  2. In Kodi, go to Add-ons
  3. Click the package installer icon (top left)
  4. Choose "Install from zip file"
  5. Navigate to the downloaded zip file and select it

Available Add-ons

  • GTV Skin - A modern Kodi skin inspired by Google TV's interface
  • GTV Helper - Helper script for the GTV skin with additional functionality

Repository Contents

  • addons.xml - Repository addon list
  • addons.xml.md5 - MD5 checksum
  • repository.zero0qrepo/ - Repository addon files
  • script.gtv.helper/ - GTV Helper addon files
  • skin.gtv/ - GTV Skin files
